# 📐 Image Area Calculator (Non-Geometric Shapes)
**[ English | [Türkçe](#-türkçe) | [العربية](#-العربية) ]**

A Python-based desktop application built with **PyQt5** and **OpenCV** that calculates the real-world area of arbitrary, non-geometric shapes from an image using a reference object of known size.

## ✨ Features
* **Modern UI:** User-friendly interface designed with PyQt5.
* **OpenCV Integration:** Powerful image processing for contour detection and area calculation.
* **Three Detection Modes:**
  * 🤖 **Semi-Automatic:** Automatically detects the largest contour as the target object and the second-largest as the reference object.
  * 🖱️ **Manual:** Allows the user to manually click and select the target object and the reference object contours.
  * ⚡ **Full-Automatic:** Experimental mode for fully automated reference detection and calculation.
* **Real-time Visualization:** Displays the threshold and contour detection results in resizable OpenCV windows.
